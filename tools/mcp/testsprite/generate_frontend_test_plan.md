# mcp0_testsprite_generate_frontend_test_plan

Generate test plan for frontend testing.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `projectPath` | string | Yes | Absolute path of project root |
| `needLogin` | boolean | No | Include login in test plan (default: true) |

## Related Tools

- `mcp0_testsprite_bootstrap_tests` - Must run first
- `mcp0_testsprite_generate_code_and_execute` - Generate and run tests
- `mcp0_testsprite_rerun_tests` - Re-run tests
