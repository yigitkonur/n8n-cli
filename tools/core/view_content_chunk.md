# view_content_chunk

View a specific chunk of a web or knowledge base document.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `document_id` | string | Yes | ID of the document |
| `position` | integer | Yes | Position of the chunk to view |

## Prerequisite

The DocumentId must have already been read by `read_url_content` before using this tool.
