# read_url_content

Read content from a URL.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `Url` | string | Yes | HTTP or HTTPS URL to read |

## Requirements

- URL must be HTTP or HTTPS
- Must point to valid internet resource accessible via web browser

## Related Tools

- `view_content_chunk` - View specific chunk of already-read document
- `mcp2_scrape_links` - Batch URL extraction with more features
