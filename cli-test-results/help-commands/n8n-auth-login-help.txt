
n8n v1.6.0

Usage: n8n auth login [options]

Configure n8n credentials

Options:
  -H, --host <url>     n8n instance URL (e.g., https://your-n8n.com)
  -k, --api-key <key>  n8n API key
  -i, --interactive    Use interactive mode with prompts
  --json               Output as JSON
  -h, --help           Display help
