
n8n v1.6.0

Usage: n8n workflows autofix [options] <id>

Auto-fix workflow validation issues

Options:
  --dry-run             Preview fixes without applying (default: true)
  --confidence <level>  Minimum confidence: high, medium, low (default:
                        "medium")
  -s, --save <path>     Save fixed workflow locally
  --apply               Apply fixes (to file or n8n server)
  --force, --yes        Skip confirmation prompts
  --no-backup           Skip creating backup before changes
  --json                Output as JSON
  -h, --help            Display help
